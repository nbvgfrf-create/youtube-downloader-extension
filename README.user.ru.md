# YouTube Downloader: инструкция для пользователя

Это приложение добавляет кнопку скачивания на YouTube.

Вам не нужно вручную устанавливать `ffmpeg` или настраивать скачивание видео. Проект старается сделать всё сам при первом запуске.

## Что понадобится

- Windows
- браузер: Google Chrome, Microsoft Edge, Яндекс Браузер, Brave, Chromium или другой Chromium-браузер
- интернет

## Самый простой запуск

### Шаг 1. Скачайте проект

Скачайте проект на компьютер и распакуйте его в обычную папку.

### Шаг 2. Запустите helper

1. Откройте папку проекта.
2. Дважды нажмите `run_helper.cmd`.

Что произойдёт:

- если Python уже есть, helper сразу продолжит настройку;
- если Python нет, скрипт сначала попробует `winget`, а затем официальный установщик Python с python.org;
- затем helper сам установит нужные библиотеки;
- затем helper подготовит встроенный `ffmpeg`;
- после этого helper запустится в фоне.

Если всё прошло успешно, окно можно закрыть.

Если Windows заблокирует оба автоматических способа, установите Python 3.11 вручную и снова запустите `run_helper.cmd`.

## Как подключить расширение

### Google Chrome

1. Нажмите в адресной строке:

```text
chrome://extensions
```

2. Включите `Режим разработчика`.
3. Нажмите `Загрузить распакованное расширение`.
4. Выберите папку `extension` внутри проекта.

### Microsoft Edge

1. Нажмите в адресной строке:

```text
edge://extensions
```

2. Включите `Режим разработчика`.
3. Нажмите `Загрузить распакованное расширение`.
4. Выберите папку `extension`.

### Яндекс Браузер

1. Откройте страницу расширений браузера.
2. Включите режим разработчика.
3. Нажмите кнопку загрузки распакованного расширения.
4. Выберите папку `extension`.

### Brave / Chromium

1. Откройте страницу расширений.
2. Включите режим разработчика.
3. Нажмите `Load unpacked` или `Загрузить распакованное расширение`.
4. Выберите папку `extension`.

## Как пользоваться

1. Запустите helper через `run_helper.cmd`, если он ещё не запущен.
2. Откройте YouTube.
3. Откройте любое видео.
4. Обновите страницу, если кнопка ещё не появилась.

После этого:

- кнопка `Скачать` под видео начнёт загрузку по настройкам по умолчанию;
- кнопка `Качество` откроет список доступных вариантов;
- через иконку расширения можно открыть popup со списком текущих загрузок и настройками.

## Как поменять язык

1. Нажмите на иконку расширения.
2. Нажмите `Настройки`.
3. На полной странице настроек helper выберите `Русский` или `English`.

Popup, кнопки на YouTube и страница настроек helper будут использовать выбранный язык.

## Как включить или выключить автозапуск

Через popup расширения:

1. Нажмите на иконку расширения.
2. Найдите переключатель `Автозапуск helper`.
3. Включите или выключите его.

По умолчанию автозапуск включён.

## Если что-то не работает

### Кнопка не появилась

1. Убедитесь, что helper уже запущен.
2. Обновите вкладку YouTube.
3. Проверьте, что расширение загружено именно из папки `extension`.

### Пишет, что helper не отвечает

Снова запустите:

```text
run_helper.cmd
```

### Видео просит cookies или авторизацию

1. Нажмите на иконку расширения.
2. Нажмите `Настройки`.
3. В настройках helper включите cookies или укажите `cookies.txt`.

Некоторым видео на YouTube cookies нужны, даже если другие видео скачиваются без них. Это нормальное поведение.

### Яндекс Браузер

Если скачивание упирается в авторизацию, лучше использовать `cookies.txt`, потому что прямое чтение cookies из Яндекс Браузера поддерживается не всегда.

## Что не нужно делать вручную

Обычно не нужно:

- искать `ffmpeg.exe`
- искать `yt-dlp.exe`
- запускать Python-команды вручную

## Где хранятся файлы

- ваши загрузки: папка, указанная в настройках helper
- настройки helper:

```text
%LOCALAPPDATA%\YouTubeDownloader\config.json
```

## Если хотите просто переустановить всё заново

1. Закройте helper.
2. Удалите папку `.venv` внутри проекта.
3. Снова запустите `run_helper.cmd`.

Скрипт заново установит зависимости и запустит helper.
